# Car Sales Analysis Dashboard

## Overview

This project is a Power BI dashboard developed to analyze and visualize car sales data in an interactive and easy-to-understand format. The dashboard provides insights into sales performance, revenue trends, pricing analysis, and manufacturer performance using Business Intelligence techniques.

---

# Technologies Used

- Microsoft Power BI  
- Microsoft Excel  
- Power Query  
- DAX  
- Data Visualization  

---

# Project Files

| File Name | Description |
|---|---|
| `BE_sagarpandit.pbix` | Main Power BI dashboard file |
| `CS_B48_SagarPandit.xlsx` | Dataset used for analysis |

---

# Dashboard Features

- Sales Performance Analysis  
- Revenue Tracking  
- Manufacturer-wise Comparison  
- Sales Trend Analysis  
- Vehicle Price Analysis  
- Engine and Horsepower Analysis  
- Interactive Charts and KPIs  
- Filters and Slicers  

---

# Key Insights

- Top-performing manufacturers identified through sales analysis.  
- Revenue comparison across different car brands.  
- Analysis of pricing and sales trends.  
- Relationship between engine size and horsepower visualized.  
- Monthly and yearly sales trends analyzed.

# Conclusion

The Car Sales Analysis Dashboard demonstrates how Microsoft Power BI can be used to transform raw automobile sales data into meaningful business insights. The project helped in understanding data visualization, dashboard development, and Business Intelligence concepts through interactive reports and analytical dashboards.
